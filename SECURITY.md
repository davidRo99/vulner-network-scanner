# Security Policy

## Authorized Use Only

This project is intended for authorized security testing in lab, classroom, internal, or owned environments.

Do not use this tool against public IP addresses, third-party systems, production networks, or any target without explicit permission.

## Responsible Testing Rules

- Define scope before scanning.
- Use only private or approved network ranges.
- Keep logs and results for auditability.
- Avoid aggressive scans in sensitive environments.
- Do not use the weak credential feature against systems you do not own or administer.

## Reporting Issues

If you find a bug in the project, open a GitHub issue with:

- Operating system and version
- Tool versions
- Command or mode used
- Relevant error output
- Expected behavior
- Actual behavior

Do not include real credentials, private IP plans from organizations, or sensitive scan results in public issues.
